# Typings

In this library there some useful TypeScript typings for your projects that you can use.

You can create `.d.ts` file in your project (for example `global.d.ts`) and add it via references.

### CSS-modules typings

This declare types for `.module.css` and `.m.css` files.

Usage:

```ts title="global.d.ts"
/// <reference types="@krutoo/utils/typings/css-modules" />
```

### React-SVG modules typings

This declare types for `.svg` files as React-components.

Usage:

```ts title="global.d.ts"
/// <reference types="@krutoo/utils/typings/react-svg" />
```
