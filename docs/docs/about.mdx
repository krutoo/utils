`@krutoo/utils` is a JavaScript library with some utilities for various cases and technologies.

## Installation

```bash
# In Node.js
npm add @krutoo/utils

# or in Deno
deno add npm:@krutoo/utils

# or in Bun
bun add @krutoo/utils
```

Some utilities are created for training and educational purposes.

- All utilities is written with TypeScript
- All utilities is well tested by unit- and E2E-tests
- The NPM package has no dependencies (except optional peer dependencies)
- The NPM package is Tree Shaking ready and has no side effects
