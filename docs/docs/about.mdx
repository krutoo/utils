`@krutoo/utils` is a JavaScript library with some utilities for various cases and technologies.

## Installation

```sh
# In Node.js
npm add @krutoo/utils

# or in Deno
deno add npm:@krutoo/utils

# or in Bun
bun add @krutoo/utils
```

Some utilities are created for training and educational purposes.

- Written with TypeScript
- Well tested by unit/e2e
- No dependencies (except optional peers)
- Tree Shaking ready
