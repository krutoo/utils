# `scaleRectToCover`

Takes "target" rectangle and "area" rectangle and returns rectangle that has same ratio as "target" but covers "area" by size.

Works like CSS rule `object-fit: cover`.

```ts
import { scaleRectToCover } from '@krutoo/utils';

const target = { width: 10, height: 10 };
const area = { width: 100, height: 20 };

scaleRectToCover(target, area); // { width: 100, height: 100 }
```

This function is useful for example when you need to scale some element on the page to cover other element with saving the original ratio.
