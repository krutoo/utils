# `Vector2`

export const meta = {
  title: 'Vector2',
  category: 'Math',
};

Class that represents a point in a 2-dimensional space.

```js
import { Vector2 } from '@krutoo/utils';

const vector = new Vector2(10, 20);

vector.setX(20);

vector.setY(20);

vector.add(new Vector2(2, 3));

vector.subtract(new Vector2(4, 5));

vector.scale(10);

vector.normalize(10);

vector.getLength(); // number

vector.getDistance(new Vector2(0, 0)); // number

vector.equalsTo(new Vector2(0, 0)); // boolean
```

Most ot the methods are _chainable_:

```js
const vector = new Vector2(10, 20);

vector
  .setX(20)
  .setY(20)
  .add(new Vector2(2, 3))
  .subtract(new Vector2(4, 5))
  .scale(10)
  .normalize(10)
  .clone();
```

Most of methods are _mutable_ so it changes the instance.

But you can use `close()` to work in immutable manner.
