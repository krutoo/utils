# `shuffle`

export const meta = {
  title: 'shuffle',
  category: 'Math',
};

Randomly sorts given array and returns it. This function mutates given array.

```tsx
import { shuffle } from '@krutoo/utils';

const array = [1, 2, 3, 4, 5, 6, 7];

// mutates given array
shuffle(array);
```

If you need to specify `random()` function you can use `options` argument:

```tsx
shuffle(array, {
  random: myCustomRandomizer,
});
```

### `toShuffled`

`toShuffled` works exactly like `shuffle` but don't mutates given array, instead it returns new randomly sorted array.

```tsx
import { toShuffled } from '@krutoo/utils';

const array = [1, 2, 3, 4, 5, 6, 7];

const newArray = toShuffled(array); // same items but randomly sorted
```
