# `randomFloat`

export const meta = {
  title: 'randomFloat',
  category: 'Math',
};

Returns float random value between `min` and `max`.

```ts
import { randomFloat } from '@krutoo/utils';

randomFloat(0.2, 0.5); // float number in range 0.2-0.5
```

Under the hood the `Math.random()` is used.

If you want to create you own `randomFloat` with custom `random` function you can use `createRandomFloat`:

```ts
import { createRandomFloat } from '@krutoo/utils';

// customRandom should return number in range from 0 to 1
const = randomFloat = createRandomFloat(customRandom);

randomFloat(0.2, 0.5); // float number in range 0.2-0.5
```
