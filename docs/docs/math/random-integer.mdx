# `randomInteger`

export const meta = {
  title: 'randomInteger',
  category: 'Math',
};

Returns integer random value between `min` and `max`.

```ts
import { randomInteger } from '@krutoo/utils';

randomInteger(10, 60); // integer number in range 10-60
```

Under the hood the `Math.random()` is used.

If you want to create you own `randomInteger` with custom `random` function you can use `createRandomInteger`:

```ts
import { createRandomInteger } from '@krutoo/utils';

// customRandom should return number in range from 0 to 1
const = randomInteger = createRandomInteger(customRandom);

randomInteger(10, 60); // integer number in range 10-60
```
