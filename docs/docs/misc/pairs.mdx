# `pairs`

export const meta = {
  category: 'Misc',
  title: 'pairs',
};

Returns generator that iterates over each pair in array.

```tsx
import { pairs } from '@krutoo/utils';

const list = [1, 2, 3];

for (const [a, b] of pairs(list)) {
  console.log(a, b);
}

// output:
// 1 2
// 1 3
// 2 3
```
