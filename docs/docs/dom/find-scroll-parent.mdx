# `findScrollParent`

export const meta = {
  title: 'findScrollParent',
  category: 'DOM',
  menuPriority: -100,
};

Returns closest scrollable ancestor or null if there is no scrollable ancestors.

> This is a combination of `findAncestor` and `isScrollable`, it exists only for backward compatibility.

## Usage

```tsx
import { findScrollParent } from '@krutoo/utils';

const closestScrollable = findScrollParent(document.querySelector('#something'));
```
